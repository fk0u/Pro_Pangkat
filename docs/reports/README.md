# Reports Index

Folder ini berisi laporan historis implementasi, bugfix, dan optimasi.

## Konvensi Nama
- Gunakan format: `YYYY-MM-DD-topic-status.md` untuk laporan baru.
- Untuk laporan lama yang dipindah, pertahankan nama asli sampai ada harmonisasi.

## Tujuan
- Menjaga jejak perubahan teknis.
- Memudahkan audit fase implementation dan verification.
