# Archive

Folder ini menyimpan file snapshot/backup non-runtime.

## Snapshot
- Lokasi: `archive/snapshots/`
- Isi: salinan file `.new`, `.bak`, `.backup`, atau varian eksperimen yang tidak aktif.

## Aturan
- Jangan import file dari folder archive ke source runtime.
- File di archive hanya untuk referensi/rollback manual.
